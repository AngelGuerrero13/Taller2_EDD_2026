Nombre programa: MP3

Integrantes: Ángel Guerrero, Diego Godoy, Felipe Argomedo

Descripcion: Aplicación de consola diseñada para digitalizar la experiencia de escuchar música mediante archivos MP3. El software carga una base de datos de canciones y un archivo de estado para retomar la sesión anterior del usuario. Permite navegar entre pistas, gestionar una lista de reproducción dinámica y realizar operaciones de mantenimiento como agregar o eliminar canciones del registro permanente.
